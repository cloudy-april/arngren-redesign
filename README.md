# Arngren Redesign

A modern, clean e-commerce platform redesign for Arngren.net with a sleek interface and comprehensive product catalog.

## Features

- **Smart Search**: Autocomplete suggestions as you type
- **Product Catalog**: 33+ products across 5 categories (Drones, Electric Vehicles, RC & Toys, Tools, Cameras)
- **Advanced Filters**: Filter by category, price range, power type, and stock status
- **Quick View Modal**: Preview product details without leaving the page
- **Shopping Cart**: Add items, manage quantities, and checkout
- **Wishlist**: Save favorite products for later
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Offline Support**: All data stored locally with localStorage

## Technologies

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with glassmorphism effects
- **Vanilla JavaScript** - No frameworks, lightweight and fast
- **Unsplash API** - Real product images

## Pages

- **index.html** - Homepage with search bar and catalog
- **product.html** - Product detail page
- **cart.html** - Shopping cart and checkout
- **app.js** - Shared application logic
- **styles.css** - All styling

## Color Scheme

## Getting Started

1. Clone the repository
2. Open `index.html` in a web browser
3. Start searching and browsing products

## Usage

- **Search**: Type in the main search bar to find products by name or category
- **Browse**: Click category links or search results to see the full catalog
- **Filters**: Use sidebar filters to narrow down results (desktop only)
- **Add to Cart**: Click the "+ Add" button on any product
